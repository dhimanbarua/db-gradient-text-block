== Installation ==
From within WordPress' dashboard:

1. Go to Plugins -> Add New
2. Search for "DB Gradient Text Block"
3. Click "Install"
4. Click "Activate"
5. New Gutenberg Blocks will appear!


== Frequently Asked Questions ==
= How to insert blocks of DB Gradient Text in your page or post? =
Install and activate DB Gradient Text for Gutenberg on your website. Create a new page and click the ‘+’ sign, and you will find all the creative Gutenberg blocks there. Now click on the block you want to insert in your page and create amazing pages.
